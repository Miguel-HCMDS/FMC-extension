# FMC-extension
Extensão do Chrome para enviar propostas mais facilmente e encontrar palavras chave sobre design.

A extensão tem diversas caixas de seleção que inserem 3 tipos de texto diferentes e 10 tipos de serviços 
ofertados por mim além de um link para trabalhos meus atrelados ao tipo de serviço.

<img width="485" height="577" alt="Captura de tela 2026-01-07 202128" src="https://github.com/user-attachments/assets/03c9fb76-ed16-44e1-a8e5-41db6c82347b" />

Clicando na seta na navbar você vai para a sessão de achar da extensão e pode achar varias palavras chave na página, clicando na seta novamente você volta para a criação de templates.
